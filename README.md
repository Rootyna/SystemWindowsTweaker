## О проекте

System Windows Tweaker — инструмент для тонкой настройки и оптимизации Windows 10 и 11.

Создан как безопасная альтернатива сторонним утилитам, которые часто вносят изменения без контроля и предупреждений.

Инструмент работает на уровне системного реестра, служб и параметров конфигурации, поэтому требует осторожного использования.

**Рекомендация:** Перед применением изменений создайте точку восстановления системы  
(Win + R → `systempropertiesprotection` → диск C: → «Создать»).

Автор не несёт ответственности за возможные последствия некорректного использования.

- Пользователи, желающие ускорить систему без установки тяжёлых сторонних программ
- Энтузиасты, предпочитающие контроль над изменениями
- Администраторы, ищущие готовый набор проверенных оптимизаций

---

## Функциональность

| Категория                  | Основные возможности                                                                 |
|----------------------------|--------------------------------------------------------------------------------------|
| **Производительность**     | Отключение ненужных служб, телеметрии, визуальных эффектов                           |
| **Конфиденциальность**     | Блокировка сбора данных Microsoft, удаление встроенных приложений                    |
| **Очистка**                | Удаление временных файлов, кэша, мусора из системы                                   |
| **Интерфейс**              | Настройка меню Пуск, панели задач, Explorer                                         |
| **Дополнительно**          | Оптимизация планировщика задач, управление UAC, настройка контекстного меню         |

Все изменения применяются только после подтверждения пользователя.

## Запуск

### Требования
- Windows 10 (версия 20H2 и выше) или Windows 11
- Права администратора

### Инструкция
1. Скачать репозиторий или основной файл `SWT_RT.bat`
2. Запустить файл двойным щелчком или через контекстное меню («Открыть»)

Скрипт автоматически проверит права и архитектуру системы.

---

## Безопасность

- Все критические изменения требуют явного подтверждения
- Ведётся подробный лог операций
- Возможность отмены большинства изменений через встроенные пункты
- Код открыт и доступен для проверки
- Не содержит внешних загрузок или подозрительных действий

При возникновении проблем создавайте issue в репозитории — помощь будет оказана оперативно.
